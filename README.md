# TodoListApp-Typescript

React Native Todo List App With ngrok 

## Getting Started

Clone repo 

### Prerequisites
Already installed ngrok (https://ngrok.com/)

### Installing

     npm install
 
## Running 
need 3 command line to start this project


cd/JsonServer run JsonServer

    npm start db
   
run ngrok cmd

    ngrok http 3000
  
run expo

    expo start
    
run emulator android or ios
